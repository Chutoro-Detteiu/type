# benesse-type.py

[ベネッセのタイピングサイト](https://manabi.benesse.ne.jp/gakushu/typing/)にて自動タイピングを行う。

## 必要な物

* このプログラム
* python3系
* pyautogui
* selenium
* [chromedriver-binary](https://pypi.org/project/chromedriver-binary/)
